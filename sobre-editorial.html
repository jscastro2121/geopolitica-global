Guia Editorial — Geopolítica Global

Fluxo completo de publicação automatizada

Atualizado em 26 de outubro de 2025

Este guia apresenta o fluxo editorial do projeto Geopolítica Global — da escrita do texto até a publicação automatizada no site. Ele documenta todas as etapas do sistema que integra conteúdo, design e automação via Python.

1. Estrutura de diretórios

geopolitica-global/
¦
+-- artigos/             ? artigos gerados pelo script
+-- colaboracoes/        ? colaborações de leitores
+-- textos/              ? arquivos .txt originais
+-- includes/            ? header.html e footer.html
+-- css/                 ? main.css, artigo.css, responsivo.css
+-- js/                  ? ativo.js e menu.js
+-- imagens/             ? capas e ícones
+-- index.html           ? página inicial (atualizada automaticamente)
+-- gerar_artigo.py      ? script gerador

2. Escrita do texto base

Os artigos e colaborações devem ser escritos em texto puro (.txt), com parágrafos separados por linhas em branco. O script transforma automaticamente esse texto em HTML formatado.

Dica: evite qualquer tag HTML. Use apenas o texto e, se quiser destacar algo, adicione uma citação com o símbolo “>”. 
### 1. O Custo da Mentira

> A verdade tornou-se cara demais.

Enquanto se discute moralidade, a economia segue guiada pela conveniência.

3. Geração automática do HTML

No terminal, estando na raiz do projeto, execute:

python gerar_artigo.py

Responda às perguntas indicadas:

Tipo (artigo/colaboracao): artigo
Título: A Hipocrisia como Sistema
Subtítulo: Reflexão sobre a dissimulação na vida pública brasileira
Autor: Santana
Data (ex: 26 de outubro de 2025): 26 de outubro de 2025
Nome do arquivo (sem espaços, ex: hipocrisia): hipocrisia
Caminho do texto base (.txt): textos/hipocrisia.txt

O script criará o arquivo HTML, copiará o texto original para /textos/ e atualizará automaticamente o index.html, sem duplicar links já existentes.

4. Estrutura do HTML gerado

O arquivo resultante segue o padrão dos artigos principais:

<!--#include virtual="includes/header.html" -->
<script src="/js/ativo.js"></script>

<main class="duas-colunas">
  <article class="col-esquerda">
    <section class="artigo">
      <h2>[TÍTULO]</h2>
      <h3>[SUBTÍTULO]</h3>
      <p class="meta">Publicado em [DATA]</p>
      <hr>
      [CONTEÚDO]
    </section>
  </article>
  [ASIDE AUTOMÁTICO]
</main>

5. Atualização do index.html

O sistema insere automaticamente o novo artigo na seção correta:

<article class="card-artigo">
  <h3><a href="/artigos/hipocrisia.html">A Hipocrisia como Sistema</a></h3>
  <p>Reflexão sobre a dissimulação na vida pública brasileira.</p>
</article>

Se o link já existir, o script detecta e evita duplicação.

6. Boas práticas editoriais

  • Use nomes de arquivo em minúsculas e sem espaços (apartheid-financeiro.txt).
  • Salve os arquivos em UTF-8 para evitar problemas com acentuação.
  • Use apenas linhas em branco para separar parágrafos.
  • Evite duplo “.txt.txt” no nome do arquivo.

7. Revisões e ajustes

Para editar um artigo:

  1. Edite o arquivo original em /textos/;
  2. Execute novamente o script com os mesmos parâmetros;
  3. O HTML será atualizado sem criar duplicatas no index.html.

8. Futuras expansões

O sistema permite integração futura com:

  • remoção automática de artigos via terminal;
  • geração de índice cronológico;
  • exportação para PDF ou Substack;
  • capas automáticas com imagem padrão.

Conclusão

Com esse fluxo, o projeto Geopolítica Global atinge autonomia editorial plena: você escreve em texto puro, executa o gerador e publica com consistência visual e técnica.

Sobre o Projeto

O Geopolítica Global é uma publicação independente dedicada à análise das forças políticas e econômicas que moldam o século XXI.
